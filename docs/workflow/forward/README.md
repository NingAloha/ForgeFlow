# Forward Criteria

这里收主流程正向推进条件。

核心语义：

* `ready(current artifact) -> execute(next agent) -> produce(next artifact)`
* 调度决策中的 `next_stage_to_execute` 表示“下一步要执行的 agent 阶段”，不是“已完成的阶段状态”。
