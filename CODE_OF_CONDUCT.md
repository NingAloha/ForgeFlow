# 行为准则（Code of Conduct）

English: `CODE_OF_CONDUCT.en.md`

## 我们的承诺

作为成员、贡献者与维护者，我们承诺让所有人都能在社区中获得免于骚扰的参与体验，
不因年龄、体型、可见或不可见的残障、民族、性别特征、性别认同与表达、经验水平、
教育背景、社会经济地位、国籍、外表、种族、宗教、性取向与性别取向而受到区别对待。

我们承诺以促进开放、友善、多元、包容与健康社区的方式行动与互动。

## 我们的标准

积极行为示例包括：

- 对他人保持同理心与友善
- 尊重不同意见、观点与经历
- 提供并欣然接受建设性反馈
- 为自己的错误承担责任，向受影响者道歉，并从中学习
- 关注对整个社区最有利的事情

不可接受行为示例包括：

- 使用性暗示语言或图像，以及性关注或性挑逗
- 挑衅、侮辱或贬损性评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人私人信息（如住址或邮箱）
- 其他在专业环境中可被合理认为不适当的行为

## 执行责任

维护者负责澄清与执行可接受行为标准，并对不当、威胁、冒犯或有害行为采取适当且公平的纠正措施。

维护者有权也有责任删除、编辑或拒绝与本行为准则不一致的评论、提交、代码、wiki 编辑、issue
及其他贡献，并在适当时说明原因。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共场合代表本社区时的行为。

## 执行

如遇滥用、骚扰或其他不可接受行为，请联系项目维护者。所有投诉将被及时且公平地审查与调查。
维护者将尊重并保护举报人的隐私与安全。

报告渠道（任选其一）：

- **首选（私密）**：通过 GitHub Security Advisories（repo 的 **Security** tab）发起一条 private
  report（用于私密沟通与跟进）。
- **备选（公开，但请勿泄露敏感信息）**：提交一个 GitHub issue，并在正文中只写最少必要信息；
  维护者会在需要时转到私密渠道继续沟通。

## 执行指南

维护者将参考以下“社区影响指南”来决定处置措施：

### 1. 纠正

**社区影响**：使用不当语言或其他被认为不专业或不受欢迎的行为。

**后果**：维护者私下书面警告，说明违规性质与原因；可能要求公开道歉。

### 2. 警告

**社区影响**：单次事件或一系列行为造成的违规。

**后果**：带有后续违规后果的警告；在指定时间内不得与相关人员互动。

### 3. 临时封禁

**社区影响**：严重违规，包括持续不当行为。

**后果**：在指定时间内禁止与社区进行任何形式的互动或公开沟通。

### 4. 永久封禁

**社区影响**：模式化违规，包括持续不当行为、针对个人的骚扰，或对群体的攻击与贬损。

**后果**：永久禁止在社区内进行任何形式的公开互动。
